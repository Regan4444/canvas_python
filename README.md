# canvas_python
A repository of python scripts used to audit Canvas LMS
Script #1 = canvas_respondus_audit = audits all copied classes within a specified marking period to determine if Respondus LockDown Browser has been activate for the class and that the link to LDB has been opened to verify that all copied settings have been transferred.
This is a very eclectic collection.  Most if not all scripts have comments explaining their use.  In every instance you will need to paste in your own domain ( ex. https://school.instructure.com) as well as your own access token.  You will need to be a domain level admin to run these scripts
